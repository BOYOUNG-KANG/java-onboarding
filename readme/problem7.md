## 기능 수행 이해
1. 입력값 받기
    - [ ] user 입력값 받기
      - [ ] 1이상 30 이하 문자열
      - [ ] 예외 발생 시, IllegalArgumentException 발생
    - [ ] friends 입력값 받기
      - [ ] 길이가 1이상 10000 이하
      - [ ] 각 배열 길이가 2
      - [ ] 각 아이디 길이가 1이상 30 이하 문자열
      - [ ] 예외 발생 시, IllegalArgumentException 발생
    - [ ] visitors 입력값 받기
      - [ ] 0 이상 10000 이하 리스트(배열)
      - [ ] 예외 발생 시, IllegalArgumentException 발생
2. 친구 추천 계산하기
   - [ ] 함께 아는 친구 계산하기
     - [ ] 함께 아는 친구의 경우 +10
   - [ ] 타임라인 방문자 계산하기
     - [ ] 방문자의 경우 +1
3. 추천 친구 출력학기
   - [ ] 가장 높은 추천 친구 5명 출력
     - [ ] 추천 점수가 0일 경우, 추천하지 않음
     - [ ] 추천 점수가 같은 경우, 이름순 정렬
